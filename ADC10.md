# Introduction #

The ADC10 library allows you to access the integrated ADC10 included in some devices.



# Details #

## Device Support ##

The ADC10 Library supports the following families:
  * Value Line (G2xx)
  * F2 Series (F2xx)
  * F5 Series (F5xx)
